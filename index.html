MafiaRP Documentation

Welcome to the documentation for the MafiaRP Schema.

The MafiaRP Schema is a foundational template designed to support immersive role-playing experiences within a mafia-themed environment. It provides essential functionalities and a robust structure that can be customized and expanded to meet the specific needs of your game server.

To utilize this schema, ensure you have the Lilia Framework installed.
